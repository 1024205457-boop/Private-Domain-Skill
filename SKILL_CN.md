# 私域推送运营技能

## 概述

本技能管理端到端的私域内容推送流水线：
1. **文案生成** — AI 生成推广文案（智谱AI）
2. **图片生成** — AI 生成推广图片（ModAI/豆包）
3. **任务配置** — 自动化多渠道推送配置（Emon RPA）

---

## 阶段一：文案生成

### 主命令

```bash
python generate_all.py --start MMDD --days N [--format-prompt "调整要求"]
```

### 产品线

| 产品 | 渠道 | 推送频率 | AI 模型 |
|------|------|----------|---------|
| 小课包 | 群发、朋友圈、私信 | 每天 / 私信仅周六 | glm-4-flash |
| 会员 | 群发、朋友圈、私信 | 每天 / 私信仅周二 | glm-4-flash |

### 小课包文案规则

**主题循环（14个主题轮转）：**
蘑菇 → 火星 → 月球 → 钟表 → 建筑里的神兽 → 有毒生物 → 伪装大师 → 非洲动物 → 故宫 → 敦煌 → 钱币 → 四大发明 → 极地动物 → 长江黄河

**文案模板：**
```
{主题emoji} 0.1元小课包上新｜{AI生成标题}
{链接1}
✨ {卖点1}  (10-14字)
🎯 {卖点2}  (10-14字)
🎬 {卖点3}  (10-14字)

🧧另享40+主题，400集3D动画，59元拥有：
{链接2}
```

**生成规则：**
- 标题：8-14字，主题相关，禁止通用表述
- 卖点：基于真实课程内容（THEME_MATERIALS），经事实核查
- emoji 前缀：从 ✨🎯🎬📚🌟🔬💡🎪 随机不重复选取
- 装饰框：30% 概率（启用时减少为2个卖点）
- 去重：与历史标题和卖点对比

### 会员文案规则

**大类循环（11个大类）：**
生物 → 天文 → 物理 → 化学 → 数学 → 人体 → 历史 → 文学 → 地理 → 科技 → 建筑

**文案模板：**
```
📚【{大类}】百科科普课堂
{开头语}{AI生成知识点}

斑马百科会员首月仅需5️⃣9️⃣元
{链接}
🌟{引导语}
```

**生成规则：**
- 知识点：约34字，2-3句话
- 开头语：从 [你知道吗？/ 你注意过吗？/ 你发现了吗？/ 有个秘密，/ 涨知识啦！] 随机
- 引导语：从 [带孩子探索更多奥秘 / 陪孩子发现世界的精彩 / ...] 随机
- 语义去重（50% 相似度阈值）
- 事实核查（glm-4-plus 二次验证）

### 输出

- Excel 文件（按版本号递增）
- 石墨表格列结构：
  - A列：日期（MM-DD 格式）
  - B列：星期几
  - C列：会员私信（仅周二）
  - D列：会员群发
  - E列：会员朋友圈
  - F列：小课包私信（仅周六）
  - G列：小课包群发
  - H列：小课包朋友圈

## 阶段二：图片生成

### 小课包群发图

**命令：**
```bash
node batch.js --all -n 5              # 所有主题，每个5张
node batch.js -t 故宫 -t 敦煌 -n 3    # 指定主题
```

**规格：**
- 尺寸：2600×2080（5:4 比例）
- 模型：doubao-seedream-4.5
- 风格：卡通人物（男孩+猴子）在主题场景中
- 文字叠加：主题名 + 副标题（部分主题）
- 输出：`output/{主题id}/{主题}_v{批次}_{序号}.jpg`

### 会员群发图

**命令：**
```bash
node generate.js -t qunfa -s 0701 -d 7 -n 3
```

**规格：**
- 尺寸：2600×2080
- 风格：人物在多样场景中（30个场景池随机）
- 固定文字："新人首月59元"、"连续包月首月直降50元"、"点我开通"
- 输出：`output/qunfa/{MMDD}/59元小程序{MMDD}_{序号}.jpg`

### 会员朋友圈图

**两步生成：**

**第一步：生成背景**
```bash
node generate.js -t pyq -s 0701 -d 7 -n 3
```
- 尺寸：2048×3072（2:3 竖版）
- 纯风景背景 + 顶部文字

**第二步：合成最终图**
```bash
node compose-pyq.js --dir output/pyq
```
- 图层：背景 + 猴子胶片 + 权益卡 + 二维码
- 输出：`output/pyq/pyq_composed/{MMDD}/`

## 阶段三：推送配置

### 主命令

```bash
node api/配置推送.js 开始日期 结束日期 [选项]
```

**选项：**
| 参数 | 说明 |
|------|------|
| `--generate` | 先执行文案生成 |
| `--小课包` | 仅小课包 |
| `--会员` | 仅会员 |
| `--dacu` | 大促模式 |
| `--format-prompt "..."` | AI 格式调整 |

### 创建的任务类型

| 任务 | 渠道 | 频率 | 时间 |
|------|------|------|------|
| 小课包群发 | 企微群 | 每天 | 19:00 |
| 小课包朋友圈 | 朋友圈 | 每天 | 19:00 |
| 小课包私信 | 私信 | 仅周六 | 18:00/19:00/20:00 |
| 会员群发 | 企微群 | 每天 | 12:00 |
| 会员朋友圈 | 朋友圈 | 每天 | 12:00 |
| 会员私信 | 私信 | 仅周二 | 12:00/13:00/14:00 |

### 素材库

**文件夹结构：**
| 文件夹 | 内容 | 命名规则 |
|--------|------|----------|
| 群发 | 会员群发卡片 | `59元小程序-{MMDD}.jpg` |
| 朋友圈 | 会员朋友圈图 | `{MMDD}朋友圈.jpg` |
| 私信 | 会员私信卡片 | 同群发 |
| 小课包 | 小课包群发卡片 | `{主题}-{MMDD}.png` |
| 小课包朋友圈 | 小课包朋友圈图 | `{主题}·百科朋友圈.jpg` |

**重要规则：**
- 群发/私信任务使用素材库中的小程序卡片 → 任务创建前必须先更新素材库
- 朋友圈任务直接上传图片 → 不需要更新素材库
- 素材库更新命令：`node 素材库更新-api.js --qunfa/--sixin/--dacu`

## 常用操作

### 每周例行

```bash
# 1. 生成下周文案（周一到周日）
python scripts/generate/generate_all.py --start 0707 --days 7

# 2. 生成图片（如有新主题或新日期）
cd image-generation/小课包群发图 && node batch.js -t 故宫 -n 5
cd image-generation/会员素材 && node generate.js -t qunfa -s 0707 -d 7

# 3. 一键配置所有任务
node emon/api/配置推送.js 0707 0713 --generate
```

### 一键全流程

```bash
node emon/api/配置推送.js 0707 0713 --generate
```

此命令会依次执行：
1. 生成所有文案（内部调用 Python）
2. 写入石墨表格
3. 更新素材库
4. 创建所有群发/朋友圈/私信任务
5. 发送企微 webhook 通知

### 紧急操作

```bash
# 停止运行中的任务
node emon/停止任务.js "任务名关键词"

# 检查账号在线状态
node emon/账号离线检查.js

# 单独更新素材库
node emon/素材库更新-api.js --qunfa 0707
```

|------|------|

## 约束与规则

1. **私信排期**：小课包私信 = 仅周六；会员私信 = 仅周二
2. **主题轮转**：14个主题固定顺序循环，进度文件记录位置
3. **去重机制**：AI 内容需与全量历史对比后才接受
4. **事实核查**：所有知识点由二级 LLM 验证准确性
5. **素材顺序**：素材库必须在创建任务前更新
6. **朋友圈独立**：朋友圈任务直传图片，不走素材库
7. **日期格式**：石墨写入 `MM-DD` 字符串（不是整数，不是 YYYY-MM-DD）
8. **图片命名**：各文件夹有严格命名规范（见素材库表格）
9. **禁止停止朋友圈任务**：配好后不再动
10. **搜索需过滤创建者**：搜索任务时必须加 creator 过滤，避免冲突

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 文字AI | 智谱AI (glm-4-flash/plus) | 文案生成 & 事实核查 |
| 图片AI | ModAI (doubao-seedream-4.5) | 图生图 |
| 后端 | Python (Flask + SocketIO) | 部署面板 & 文案生成 |
| 自动化 | Node.js | RPA API 调用 & 图片生成 |
| 数据 | Excel (openpyxl)、石墨 API | 配置与内容存储 |
| 执行 | Emon RPA 平台 | 任务实际执行 |
| 通知 | 企业微信 Webhook | 部署完成提醒 |
